
# dharmnumerology - Student Management System

A professional, cloud-synced dashboard for managing Numerology, Laal Kitab, and Vastu masterclasses.

## Features
- **Neon Cloud Integration:** Real-time data persistence using serverless PostgreSQL.
- **Masterclass Management:** Create batches, track enrollments, and manage students.
- **Accounts & Reports:** Track fees, pending dues, and export CSV reports.
- **Daily Cosmic Insights:** AI-generated daily guidance using Google Gemini.
- **Student Portal:** Access to class recordings and digital certificates.
- **Marketing Hub:** Lead generation for free webinars.

## Tech Stack
- **Frontend:** React, Tailwind CSS, Lucide Icons.
- **Database:** Neon (PostgreSQL).
- **AI:** Google Gemini API.
- **Deployment:** Vercel.

## Setup
1. Clone the repository.
2. Install dependencies.
3. Set `DATABASE_URL` for Neon connection in environment variables.
4. Set `API_KEY` for Google Gemini AI.
